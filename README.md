# About This Project

Swift Cart is a simple e-commerce website that I built using HTML and CSS.  
I created this project to practice my frontend development skills and improve my understanding of layouts and responsive design.

This project helped me understand how real e-commerce websites structure their product sections and navigation system.



# What I Used

*HTML5
* CSS3
* Tailwind CSS 
* Font Awesome 



# What I Learned

While building this project, I practiced:

- Creating responsive layouts
- Designing product cards
- Working with navigation menus
- Using Flexbox and Grid
- Deploying a website using GitHub Pages



# Why I Built This

I am currently learning web development and trying to improve my frontend skills.  
This project is one of my practice projects to become more confident in building real-world websites.
